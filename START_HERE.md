# [Personal] AI 영양제 분석기 (Antigravity 실전 프로젝트)

제이, 오늘 저녁 Antigravity와 함께 바로 시작할 수 있도록 세팅을 마쳤습니다! 💊

## 🎯 목표
사용자가 입력한 영양제들의 궁합을 AI가 분석하고, 부족한 영양소를 추천해주는 웹 서비스의 MVP(최소 기능 제품) 만들기.

## 🚀 Antigravity 시작 프롬프트
이 파일의 내용을 복사해서 Antigravity 채팅창에 바로 붙여넣으세요:

---
"나는 Next.js와 Tailwind CSS를 사용해서 'AI 영양제 분석기' 웹사이트를 만들 거야.
이 프로젝트의 핵심은 사용자가 먹는 영양제 목록을 넣으면 GPT-4o API를 통해 궁합을 분석해주는 거야.

1. **디자인 컨셉:** 모던하고 깔끔한 헬스케어 스타일 (화이트 & 민트 포인트).
2. **주요 기능:**
   - 영양제 이름을 추가할 수 있는 입력 필드와 리스트 UI.
   - '분석 시작' 버튼 클릭 시 AI API 호출 (현재는 Mock Data로 구조만 먼저 잡아줘).
   - 분석 결과를 '좋음/주의/피함'으로 구분한 카드 레이아웃.
3. **폴더 구조:** `src/app` 폴더 구조를 사용하고, 컴포넌트 중심으로 나누어줘.

자, 이제 프로젝트의 기본 뼈대 코드를 작성해줘."
---
